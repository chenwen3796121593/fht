## 一、项目

React 19 + Vite + Tailwind CSS + Supabase + Cloudflare Pages

```
src/
├── pages/           HomePage / Dashboard / IndicatorsPage / NewsPage / ChatPage / AlertsPage / VipPage
├── components/      TopBar / MarketBar / Watchlist / StockChart
├── hooks/           useMarketData / useAlertChecker
functions/api/       data / kline / breadth / flow / rss-news / macro-news / macro-data / sh-monthly / search
scripts/             update-kline.mjs (GitHub Actions 每日更新K线到KV)
public/              PWA manifest / service-worker / _headers
```

## 三、各页面功能

### 首页
8品种横滚卡片 + 两市成交额 + 市场温度计 + 涨跌家数 + 板块资金流向TOP10

### 自选
实时K线蜡烛图（手绘SVG） + 自选列表（默认显示3品种，可滑动） + 中文名搜索添加 + OHLC实时更新

### 先行指标（6.25新增）
2×2四宫格：M1/M2货币供应、新增贷款、贷款增速、准备金率。每张卡片含折线图 + 数据表 + 右轴灰色虚线上证月线叠加。每小时自动刷新，手动刷新按钮含旋转动画。

### 新闻
RSS股票/商品/宏观三栏切换，星级排序，5分钟自动更新宏观

### 聊天
Supabase实时群聊，文字+语音消息，24h历史

### 预警
涨跌幅/价格触发，推送+震动+语音三种通知方式，5分钟防骚扰冷却

### VIP专区（6.25新增）
- 申请：手机号+理由 → Supabase
- 管理员（chen/859168）：审核申请、发布每日策略
- 会员登录后：查看今日策略 + VIP专属功能列表

## 五、关键问题与解决

| 问题 | 解决 |
|------|------|
| 聊天页比别页长 | dvh自适应 |
| K线数据全错 | 换新浪 GlobalFutures |
| K线不实时更新 | 去日期检查+强制重绘+OHLC实时数据 |
| 跌停总显示0 | 换财联社API+防缓存 |
| 板块资金无数据 | 换 emdatah5 域名 |
| 自选加不上 | 中文名搜索+前缀自动补 |
| 聊天切页崩 | 恢复模块级单例 |
| 新闻排序错 | 各标签独立按星排序 |
| 宏观指标不显示 | 东方财富API格式变更，适配result.data+字段映射 |
| 指标页切标签全黑 | useEffect清理改useRef |
| 刷新按钮点不动 | 加w-7/h-7+背景色+旋转动画 |
| 预警推送/震动/语音无效 | 补navigator.vibrate+请求权限+speechSynthesis.cancel |
| 推送开关点一下锁死 | 权限拒绝不阻塞toggle |
| 自选列表超一行 | max-h 232→150px |
| 新闻刷新不转 | 补spin状态+最短600ms |
| Logo太小 | Flame 20→26px，文字text-sm→text-lg |
| 部署域名不更新 | wrangler加--branch=main，_headers禁HTML缓存 |
| VIP看不到新申请 | PWA Service Worker缓存Supabase API，排除api/和supabase.co |
| 申请重复提交 | 插入前查重，按钮防连点 |
