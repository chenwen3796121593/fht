# 烽火台 TradeBoard — 开发记录

> 2026.6.22-26 · 线上 https://fenghuotai.pages.dev

---

## 一、项目概要

七页跨市场行情看板：首页·自选·指标·新闻·聊天·预警·VIP  
React 19 + Vite 8 + Tailwind CSS 4 + Supabase + TradingView Lightweight-Charts v5 + Cloudflare Pages

---

## 二、页面功能

### 🏠 首页
7品种横滚卡片 + 两市成交额 + 市场温度计（雪花/火焰图标） + 涨跌家数（财联社） + 板块资金流向TOP10（东方财富）

### 📈 自选
TradingView lightweight-charts K线图（缩放/拖拽/十字光标） + 成交量柱状图 + 自选列表（中文名搜索/代码添加） + 分时/日线/全部切换 + localStorage缓存K线

### 📊 先行指标
2×2四宫格（M1/M2、新增贷款、贷款增速、准备金率），全部使用 lightweight-charts 折线图，上证明细灰色叠加，24h localStorage缓存

### 📰 新闻
股票/商品/宏观三栏切换，RSS星级排序，5分钟自动更新

### 💬 聊天
Supabase实时群聊 + 文字/语音消息 + Lucide风格表情面板（48个六行分组） + 乐观更新（消息即时显示） + localStorage缓存（打开秒显）

### 🔔 预警
涨跌幅/价格触发三种条件 + 推送/震动/语音通知 + 5分钟防骚扰冷却 + 品种名自动取实时行情全名

### 👑 VIP专区
- 申请：手机号+理由 → Supabase，防重复提交
- 登录：VIP用户Supabase验证，管理员后端CF Function校验（本地回退硬编码）
- 管理员：策略发布（内联反馈+Toast） + 申请审核（通过设置密码创建会员/拒绝）
- 会员：今日策略 + 功能列表

## 三、架构优化（6.26）

| 优化 | 说明 |
|------|------|
| 公共常量提取 | `src/lib/constants.js`：轮询间隔、品种列表、Supabase配置 |
| Supabase单例 | `src/lib/supabase.js`：同步初始化（去动态import，提速） |
| Hook合并 | `useMarketData` 统一提供 `prices` + `quotes`，减少重复请求 |
| React Context | `AppContext`：全局路由 + 行情数据 + Toast，消除 props drilling |
| TopBar简化 | 从7个回调props → 1个active prop，导航图标自动生成 |
| VipPage拆分 | 5个文件：入口(VipPage) + 首页(VipHome) + 登录(VipLogin) + 会员(VipMember) + 管理员(VipAdmin) |
| ErrorBoundary | 组件崩溃显示重试UI，不白屏 |
| Toast通知 | Context驱动，底部弹出，3秒消失，info/error/success三种 |
| 骨架屏 | Skeleton组件：卡片/图表/列表/新闻四种占位样式 |
| 响应式 | 移动端390px → 平板480px自适应 |
| PWA增强 | 预缓存静态资源 + 离线回退 + manifest补全 |
| 管理员后端校验 | `functions/api/admin.js`：HMAC-SHA256 token，CF环境变量 |
| 手机号显示 | `server.cjs` 清理打印日志 |
| 聊天优化 | 同步Supabase + 消息缓存 + 乐观更新 + 即时语音预览 |
| 表情面板 | 48个Lucide图标，6行×8列，日常/交易/社交/生活分类 |
| 电子时钟 | TopBar 烽火台下 HH:mm:ss，橙色 → 白色 |
| 名称修复 | Sina GBK截断 → 存储名优先；XD/XR/DR前缀自动清除 |

## 四、关键问题与解决

| 问题 | 解决 |
|------|------|
| 聊天延迟 | Supabase同步import + localStorage消息缓存 |
| XD贵州茅截断 | 存储名优先，Sina名备用，XD/XR/DR自动过滤 |
| 预警名称不对 | 从Context取实时行情名称 |
| 指标/新闻本地无数据 | server.cjs补全5个代理路由 |
| 涨跌家数一直加载 | breadths路由===改为startsWith |
| lightweight-charts v5 API | addCandlestickSeries→addSeries(CandlestickSeries,{})，PascalCase导出 |
| VIP发布策略无反馈 | 内联状态文字(✅/❌) + Toast双保险 |
| 语音消息不即时 | blob URL本地预览→后台上传→订阅替换 |
| 聊天消息不显示 | 乐观更新tmp_前缀，订阅去重替换 |
| Toast不显示 | 事件总线→AppContext Provider内渲染 |
| 温度计emoji不协调 | 换lucide Snowflake/Flame图标 |
| K线数据全错 | 换新浪 GlobalFutures |
| 板块资金无数据 | 换 emdatah5 域名 |
| 预警推送无效 | 补navigator.vibrate+Notification API |
