## 一、项目概要

八页跨市场行情看板：首页·自选·指标·新闻·聊天·预警·贵金属·VIP  
React 19 + Vite 8 + Tailwind CSS 4 + Supabase + TradingView Lightweight-Charts v5 + Cloudflare Pages

---

## 二、架构设计

### Context 全局状态
- `AppContext`：路由导航、行情数据(quotes+prices)、Toast通知、涨跌数据(预加载+10s刷新)、自选品种管理
- Toast 直接在 AppProvider 内渲染，Context 驱动，无额外组件

### 懒加载 + 预加载
- Dashboard、IndicatorsPage、ChatPage、VipPage 使用 `React.lazy` 分包
- 首页渲染后 `requestIdleCallback` 后台预加载其余4个页面
- lightweight-charts(168KB) 和 Supabase(201KB) 独立分包，首屏仅222KB

### 数据流
- 本地开发：Vite → server.cjs(Node代理) → 新浪/东方财富/财联社
- 生产环境：Cloudflare Pages → CF Functions → 新浪(HTTPS)/东方财富

## 三、各页面详情

### 🏠 首页
- 7品种横滚卡片(MarketBar)，3s轮询
- 两市成交额 + 市场温度计(Snowflake/Flame图标)
- 涨跌家数：财联社 API，10s刷新，localStorage 5分钟缓存秒显（AppContext启动时预加载）
- 板块资金流向 TOP10：东方财富 emdatah5

### 📈 自选
- TradingView lightweight-charts v5 K线图(CandlestickSeries+HistogramSeries)
- 分时/日线/全部三种模式，setData 增量刷新
- 商品日K服务端限300根（避免18年数据撑爆图表），日线显示30根
- 分时K线：intraday转Unix时间戳，日线保持日期字符串
- 自选品种支持中文名搜索、代码添加、localStorage持久化
- 代码前缀自动补全(sh/sz/bj)，统一 `normalizeSymbol` 函数

### 📊 先行指标
- 2×2四宫格 SVG折线图：M1/M2货币供应、新增贷款、贷款增速、准备金率
- 右上轴叠加灰色虚线上证月线
- 东方财富 datacenter API，1h自动刷新，24h localStorage缓存

### 📰 新闻
- 股票/商品RSS(AI精选+星级排序) + 新浪宏观(关键词筛选)
- 宏观5min自动更新，股票商品按需刷新

### 💬 聊天
- Supabase Realtime(Postgres Changes 订阅)
- 48个Lucide图标表情面板(6行×8列，日常/交易/社交分类)
- 文字乐观更新(temp ID)+语音blob预览+后台上传
- localStorage消息缓存(最近30条)，打开秒显
- 消息时间显示：今天只显时分，昨天及更早显示月日+时分

### 🔔 预警
- 涨跌幅/价格突破/价格跌破三种触发条件
- 推送(Notification API)+震动(Vibration API)+语音(SpeechSynthesis)
- 5分钟防骚扰冷却，品种名自动取实时行情全名(从Context prices)

### 💰 贵金属
- **实时行情表**：沪金/沪银/铂金/钯金 销售价(卖价)，新浪期货数据(nf_AU0/AG0/PT0/PD0)，30s轮询+localStorage缓存
- **商品预测表**：Supabase commodity_rankings 实时订阅，品种/现价/7天/14天/30天五列，高于现价红色低于绿色
- flex布局：表格区可滚，底部文字+微信二维码固定不跳
- 白银元/千克→元/克自动转换

### 👑 VIP
- 拆分为 VipHome/VipLogin/VipMember/VipAdmin 四个子组件
- 管理员登录：CF Function JWT校验(环境变量 ADMIN_USER/PASS/SECRET)，本地回退硬编码
- 会员登录：Supabase vip_users 表验证
- 策略发布+申请审核(通过自动创建会员账号/拒绝)
- Toast内联反馈(✅/❌)

## 六、关键问题与解决

| 问题 | 解决 |
|------|------|
| 网页打开20秒 | index.html 去除 no-cache meta，_headers 设置 1年 immutable |
| 标签切换慢 | React.lazy 懒加载 + requestIdleCallback 预加载 |
| K线全乱 | 商品date字段兼容 + 服务端限300根 + 分时/日线时间格式分流 |
| K线每3秒闪烁 | chart.remove() → series.setData() 增量刷新 |
| 聊天发送不了 | getSB() 动态import + await + 乐观更新tmp_前缀 |
| 语音消息灰色 | blob URL → 占位文字"发送中"，订阅替换 |
| Toast不显示 | 统一走 AppContext Provider 内渲染，删除 Toast.jsx |
| 品种名截断 | normalizeSymbol + 存储名优先 |
| 预警名只有2字 | 从 Context prices 取实时行情全名 |
| 管理员密码泄露 | JWT 回退密钥移除，强制 Cloudflare 环境变量 |
| 首页白屏 | Supabase 动态 import 分包，主包 642KB→222KB |
