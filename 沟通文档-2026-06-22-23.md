## 一、项目

五页行情看板：首页·自选·新闻·聊天·预警  
React 19 + Vite + Tailwind CSS + Supabase + Cloudflare Pages

## 二、数据源

| 数据 | 来源 | 频率 |
|------|------|:--:|
| 行情 | 新浪 hq.sinajs.cn（含 OHLC） | 10s |
| K 线 | 新浪 CN_MarketData / GlobalFutures / InnerFutures | 按需+KV |
| 新闻股票 | RSS a-stocks.xml | 每日 |
| 新闻商品 | RSS commodities.xml | 每日 |
| 新闻宏观 | 新浪财经+权威筛选 | 5min |
| 涨跌家数 | 财联社 | 30s |
| 板块资金 | 东方财富 emdatah5 | 30s |
| 聊天 | Supabase | 实时 |

## 三、关键问题与解决

| 问题 | 解决 |
|------|------|
| 聊天页比别页长 | `dvh` 自适应 |
| K 线数据全错 | 换新浪 GlobalFutures（旧项目验证） |
| K 线不实时更新 | 去日期检查 + 强制重绘 + OHLC 实时数据 |
| 跌停总显示 0 | 换财联社 API + 防缓存 |
| 板块资金无数据 | 换 emdatah5 域名 |
| 自选加不上 | 中文名→代码搜索 + 前缀自动补 |
| 聊天切页崩 | 恢复模块级单例 |
| 新闻排序错 | 各标签独立按星排序 |
| 新闻不显示 | 补全 macro 初始空数组 |

## 四、架构

```
CF Pages (静态) + Functions (代理)
├── /api/data → 新浪行情
├── /api/kline → 新浪K线 (KV缓存)
├── /api/breadth → 财联社涨跌
├── /api/flow → 东方财富板块资金
├── /api/rss-news → AI精选RSS
├── /api/macro-news → 宏观筛选
└── /api/search → 中文名搜代码
GitHub Actions 每日15:30更新商品K线到KV
```

全部免费运行。

## 五、2026.6.25 新增 / 修复

### 新增：先行指标页面（第三标签）

2×2 四宫格布局，四个宏观指标卡片：
- M1/M2 货币供应（M1同比% 蓝线 + M2同比% 金线）
- 新增贷款（绿色折线）
- 贷款增速（蓝色折线）
- 准备金率（橙色阶梯线）

**数据源**：东方财富 datacenter API（`RPT_ECONOMY_CURRENCY_SUPPLY` / `RMB_LOAN` / `DEPOSIT_RESERVE`）
**上证月线叠加**：所有卡片右Y轴灰色虚线，新建 `/api/sh-monthly`（新浪日K→按月聚合收盘价）
**刷新**：每小时自动更新 + 手动刷新按钮含旋转动画 + 显示更新时间

> API 响应格式已变更为 `{version, result: {data:[]}}`，CF Function 做了路径适配和字段映射。

### 新增：VIP 专区（第七标签）

皇冠图标，位于预警右侧。包含：
- 申请 VIP：手机号 + 理由 → localStorage
- VIP 登录：用户名 + 密码 → 跳转 VIP 专区（显示预测大模型AI服务 + 专属功能列表）

### 修复

| 问题 | 解决 |
|------|------|
| 宏观指标不显示 | 东方财富 API 响应格式变化，调整 `result.data` 提取 + 字段映射 |
| 指标页切其他标签全黑 | `useEffect` 清理函数把 Promise 当函数调了，改用 `useRef` |
| 刷新按钮点不动 | 图标太小无触摸区，加 `w-7 h-7` + 背景色 + 旋转动画 |
| 预警推送/震动/语音无效 | 补全 `navigator.vibrate()` + 请求通知权限 + `speechSynthesis.cancel()` |
| 推送开关点一下锁死 | 权限拒绝后阻止了 toggle，改为不阻塞 |
| 自选列表超出一行 | Watchlist `max-h-[232px]` → `max-h-[150px]`，默认显示 3 品种 |
| 新闻刷新不转 | 补 `spin` 状态 + 最短 600ms |
| 烽火台 logo 太小 | Flame 20→26px，文字 text-sm→text-lg，h-8 对按钮手双对齐 |
| 部署后域名不更新 | 加 `--branch=main` 直写生产环境，`public/_headers` 禁 HTML 缓存 |
