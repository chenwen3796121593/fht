```
scripts/             update-kline.mjs (GitHub Actions 每日更新K线到KV)

```

## 三、各页面功能

### 首页
8品种横滚卡片 + 两市成交额 + 市场温度计 + 涨跌家数 + 板块资金流向TOP10

### 自选
实时K线蜡烛图（手绘SVG）+ 自选列表（默认3品种可滑动）+ 中文名搜索 + A股分时K线（新浪5分钟）+ 日线/全部切换

### 先行指标（6.25新增）
2×2四宫格：M1/M2货币供应、新增贷款、贷款增速、准备金率。折线图+数据表+右轴上证月线灰色虚线叠加。首次加载后24h内localStorage缓存秒开，每小时后台自动更新。

### 新闻
RSS股票/商品/宏观三栏切换，星级排序，5分钟更新宏观

### 聊天
Supabase实时群聊，文字+语音消息，24h历史

### 预警
涨跌幅/价格触发，推送+震动+语音三种通知，5分钟防骚扰冷却。点击推送按钮自动请求浏览器通知权限。

### VIP专区（6.25新增）
- 申请：手机号+理由 → Supabase
- 管理员（chen/859168）：审核申请 + 发布每日策略
- 会员登录后：今日策略 + VIP专属功能列表（2×2网格）
- 防重复提交 + 提交中状态锁定

## 四、关键问题与解决

| 问题 | 解决 |
|------|------|
| 聊天页比别页长 | dvh自适应 |
| K线数据全错 | 换新浪 GlobalFutures |
| K线不实时更新 | 去日期检查+强制重绘+OHLC实时数据 |
| 跌停总显示0 | 换财联社API+防缓存 |
| 板块资金无数据 | 换 emdatah5 域名 |
| 自选加不上 | 中文名搜索+前缀自动补 |
| 宏观指标不显示 | 东方财富API格式变更，适配result.data+字段映射 |
| 指标页切标签全黑 | useEffect清理改useRef |
| 指标页每次打开慢 | localStorage 24h缓存，瞬间渲染 |
| 刷新按钮点不动 | 加w-7/h-7+背景色+旋转动画 |
| 预警推送/震动/语音无效 | 补navigator.vibrate+请求权限+speechSynthesis.cancel |
| 推送开关点一下锁死 | 权限拒绝不阻塞toggle |
| 自选列表超一行 | max-h 232→150px |
| 新闻刷新不转 | 补spin状态+最短600ms |
| Logo太小 | Flame 20→26px，text-sm→text-lg |
| 部署域名不更新 | wrangler加--branch=main，_headers禁HTML缓存 |
| VIP看不到新申请 | PWA Service Worker缓存Supabase API，排除api/和supabase.co |
| 行情刷新慢 | 10s→3s轮询 |
| 商品分时K线 | iTick免费版无历史数据，放弃；A股分时保留（新浪scale=5） |
| K线框价格显示-- | formattedPrice回退到price |
